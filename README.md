# Docusaurus Site

Project ini adalah versi Docusaurus dari course markdown di root repo.

## Jalankan lokal

Sesuai dokumentasi resmi Docusaurus, gunakan Node.js 20+ lalu install dependency:

```bash
cd docusaurus-site
npm install
npm run start
```

Site development server akan tersedia di `http://localhost:3000`.

## Build production

```bash
cd docusaurus-site
npm run build
```

Hasil build statis akan muncul di folder `build/`.

## Catatan

- Source markdown asli di root repo tetap dipertahankan.
- Versi Docusaurus di `package.json` mengikuti dokumentasi resmi saat ini: `3.9.2`.
- Sidebar memakai mode autogenerated dari struktur folder `docs/`.
- GitHub Pages target repo: `YanHairulFirdan/go-micro-service-crash-course`
- Base URL production: `https://yanhairulfirdan.github.io/go-micro-service-crash-course/`

## Deploy ke GitHub Pages

Workflow GitHub Actions ada di `.github/workflows/deploy-docusaurus.yml`.

Sebelum deploy pertama:

1. Push repo ini ke GitHub.
2. Buka `Settings > Pages`.
3. Pada `Build and deployment`, pilih `Source: GitHub Actions`.
4. Pastikan branch default repo adalah `master`, atau ubah workflow jika memakai branch lain.
